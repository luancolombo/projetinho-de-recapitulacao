# Projetinho de Recapitulação

Esse repositório existe por um motivo bem simples: prática.

Não é um projeto pensado para produção, nem uma aplicação finalizada. A ideia aqui é usar o projeto como um espaço de prática contínua, onde eu posso testar abordagens, refatorar código, revisar conceitos e aprofundar o uso das tecnologias do ecossistema Spring Boot.

No momento, ele ainda está em andamento e deve continuar mudando conforme eu for estudando novos conceitos.

## Proposta do projeto

A proposta é ir usando esse projeto como uma base para treinar várias peças que fazem parte do dia a dia com Spring Boot, sem a pressa de transformar tudo em algo grande logo de cara.

Entre os pontos que quero praticar aqui:

- criação de endpoints REST
- organização em camadas
- tratamento de exceções
- validação de entrada
- services
- integração com banco de dados
- evolução para CRUDs mais completos
- boas práticas no Spring Boot

Também comecei a usar um frontend simples em HTML, CSS e JavaScript só para consumir os endpoints e visualizar melhor o comportamento da API durante os estudos.

## Estado atual

Hoje o projeto funciona como uma API simples com algumas operações matemáticas e uma interface básica para consumir esses endpoints.

Ainda assim, a ideia não é parar por aqui. O projeto vai continuar sendo expandido conforme eu estudar mais recursos do Spring Boot e do restante do ecossistema Java.

## Tecnologias usadas até agora

- Java
- Spring Boot
- Maven
- HTML
- CSS
- JavaScript

## Como rodar

Com o projeto aberto, basta executar a aplicação Spring Boot.

Depois disso:

- API: `http://localhost:8080`
- Frontend: `http://localhost:8080/`

## Observação

Esse projeto é, acima de tudo, um laboratório de aprendizado.

Então se alguma parte parecer simples demais, incompleta ou em refatoração, é porque esse é exatamente o objetivo: aprender fazendo.
